#  Escáner de Puertos en Bash

Un script simple de Bash para escanear un rango de puertos en un host específico. Este proyecto fue creado con fines educativos como parte de mi aprendizaje en SMR y ciberseguridad.

---

## Uso

El script no requiere instalación, solo necesita permisos de ejecución.

1.  Dar permisos de ejecución:
    bash
    chmod +x portscan.sh
    

2.  Ejecutar el escáner:
    El script necesita un argumento obligatorio
    bash
    ./portscan.sh <IP_Objetivo>
    

3.  Ejemplo:
    Para escanear los puertos
    bash
    ./portscan.sh (IP_Objetivo)
